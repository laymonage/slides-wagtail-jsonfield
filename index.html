Python's json library

Python's json library


            >>> import json
            >>> config = {'dark_mode': True, 'font_size': 2, 'color_scheme': 'pink'}
            >>> encoded = json.dumps(config)
            >>> encoded
            '{"dark_mode": true, "font_size": 2, "color_scheme": "pink"}'
            >>> decoded = json.loads(encoded)
            >>> decoded
            {'dark_mode': True, 'font_size': 2, 'color_scheme': 'pink'}
            >>> decoded == config
            True
